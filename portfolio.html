Password Cracking & Hash Analysis

Date: Jan 2025

Description: Used Python scripts to perform brute-force and dictionary attacks on weak hash algorithms.

Key Actions:

  • Generated MD5 and SHA-1 hashes for test passwords
  • Wrote a Python script to brute-force hash values using a wordlist
  • Benchmarked SHA-256 and bcrypt for stronger hashing

Sample Python Snippet:

import hashlib

wordlist = ['password', '123456', 'qwerty']
hash_to_crack = '5f4dcc3b5aa765d61d8327deb882cf99'  # MD5 for 'password'

for word in wordlist:
  hashed_word = hashlib.md5(word.encode()).hexdigest()
  if hashed_word == hash_to_crack:
    print(f'Found password: {word}')
    break

Outcome: Learned practical attacks on weak hashes and the importance of modern, computationally secure hashing algorithms.
